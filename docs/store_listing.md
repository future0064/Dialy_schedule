## Full Description (English)

**Daily Schedule** is a clean, fast, and privacy-friendly task manager that lives in your browser toolbar.

### ✅ Key Features

- **Daily task list** — start each day with a fresh list; previous days are always accessible
- **Priority levels** — mark tasks as 🔴 High, 🟡 Mid, or ⚪ Low
- **Progress bar** — see your daily completion percentage at a glance
- **History** — review the past 30 days with mini completion bars
- **Copy yesterday** — carry over unfinished tasks with one click (completed tasks are skipped)
- **Smart sorting** — sort by default order, priority, or creation time
- **Multi-language** — switch between 🇺🇸 English, 🇨🇳 中文, 🇯🇵 日本語, 🇰🇷 한국어 instantly; preference is remembered

### 🔒 Privacy

- Zero data collection — no analytics, no tracking, no servers
- All data stored locally on your device using `chrome.storage.local`
- Only one permission required: `storage`

### ☕ Support

If you find this extension useful, you can support further development via the in-extension Ko-fi link.

## Full Description (中文)

**Daily Schedule** 是一款简洁、快速、注重隐私的每日任务管理器，常驻在浏览器工具栏中。

### ✅ 主要功能

- **每日任务列表** — 每天自动生成新列表，历史记录随时可查
- **优先级** — 将任务标记为 🔴 高、🟡 中、⚪ 低
- **进度条** — 一眼看清当天完成比例
- **历史记录** — 查看过去 30 天的完成情况
- **复制昨天** — 一键将昨天未完成的任务带入今天（已完成的自动跳过）
- **智能排序** — 支持默认顺序 / 优先级 / 添加时间三种排序
- **多语言** — 支持 🇺🇸 English、🇨🇳 中文、🇯🇵 日本語、🇰🇷 한국어，语言偏好自动记忆

### 🔒 隐私保护

- 零数据收集 — 无统计、无追踪、无服务器
- 所有数据通过 `chrome.storage.local` 存储在本地设备
- 仅需一个权限：`storage`

## Screenshots Checklist (1280×800 or 640×400)

Take these 5 screenshots after loading the extension in Chrome:

1. **Today's tasks** — add 3-4 tasks with different priorities, show progress bar at ~60%

5. **Chinese UI** — switch to 中文, show full localized interface
